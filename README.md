# Walmart-Forecasting-Project
Walmart retail sales forecasting using ARIMA, Decision Tree, Random Forest, and XGBoost. Includes data cleaning, EDA, feature engineering, model training, evaluation, and visualizations to compare time‑series and machine‑learning performance.

This project analyzes Walmart retail sales data and builds forecasting models to predict future demand. It compares traditional time‑series modeling with machine‑learning approaches to determine which method provides the most accurate forecasts.

## 📊 Project Overview
- Clean and preprocess Walmart retail sales data  
- Perform exploratory data analysis (EDA)  
- Test for stationarity using the Augmented Dickey–Fuller (ADF) test  
- Build forecasting models:
  - ARIMA
  - Decision Tree Regressor
  - Random Forest Regressor
  - XGBoost Regressor
- Evaluate models using RMSE and MAE  
- Visualize predictions vs. actual sales  

## 🧰 Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Statsmodels  
- Scikit‑learn  
- XGBoost
